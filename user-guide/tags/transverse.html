Transverse tags

These tags can be used in several situation and returns data regarding the place they are caled.

<ion:count />

Counts elements like articles, medias from an article.

Usage

// Counts the number of medias <ion:count from="article" item="medias" filter="(type = 'picture' AND copyright != 'your_name') OR type=='pdf'" />

Attributes

Attribute Possible values Description
from page, article Name of the tag parent from the item to count
item String Name of the item to count
filter String PHP like filter condition. Based on the fields available for the item.
